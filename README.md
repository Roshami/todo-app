# Full Stack Todo App

## Requirements
- Docker

## Run Instructions
1. Clone this repo
2. Run: `docker-compose up --build`
3. Open http://localhost:3000

## Features
- Create tasks with title & description
- Shows only 5 most recent uncompleted tasks
- “Done” button hides completed tasks

## Tech Stack
- Frontend: React
- Backend: Node.js + Express
- Database: MySQL
